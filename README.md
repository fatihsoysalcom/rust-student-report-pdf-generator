# Rust Student Report PDF Generator

This Rust example demonstrates how to dynamically generate personalized student reports in HTML format using the Tera template engine, and then export them to high-quality PDF files using headless_chrome.

## Language

`rust`

## How to Run

1. Ensure you have Rust installed.
2. Run using Cargo's single-file execution feature: `cargo run main.rs`
3. Alternatively, create a standard Cargo project, add `tera`, `serde`, and `headless_chrome` to your Cargo.toml, and run `cargo run`.
